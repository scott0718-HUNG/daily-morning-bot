# daily-morning-bot
daily-morning-bot send picture to line
🌅 專屬自動勵志早安圖系統 (Daily Motivation Bot)壹、產品規格書 (PRD)1. 產品概述本專案為一套全自動化的早晨問候推播系統。系統將於每日早晨，自動產生一句經典、激勵人心的繁體中文勵志語錄，並透過 AI 生成符合該語境的唯美真實人物或風景背景圖，合成日期與出處後，透過 LINE 發送給使用者，開啟美好的一天。2. 核心功能AI 語錄生成：每日隨機產生不同主題的勵志語錄、出處與作者（繁體中文）。AI 圖像生成：根據語錄情境，生成高畫質的唯美真實風格背景圖。影像後製合成：左上角：動態壓上當天日期（如：2026年03月19日）。畫面中下方：文字自動換行排版語錄，並於下方標註出處與作者。雲端圖床託管：自動將合成圖片上傳至 ImgBB，取得公網圖片連結。LINE 推播通知：透過 LINE Messaging API 將圖片直接發送至使用者的 LINE 聊天室。無伺服器定時任務：利用 GitHub Actions 設定 Cron Job，實現零成本、免主機的自動化運行。3. 系統架構與技術棧程式語言：Python 3排程與執行環境：GitHub Actions文字與圖片生成：Google Gemini API (gemini-2.5-flash 文本模型, imagen-4.0-generate-001 圖像模型)圖片處理：Pillow (Python Imaging Library) + 動態下載 Noto Sans TC 開源字型。圖片圖床：ImgBB API訊息發送：LINE Messaging API貳、安裝與部署步驟說明只要跟著以下 5 個步驟，即可完成您的免費專屬早安圖系統！步驟一：取得必要的 API 密鑰 (Keys)您需要收集以下四組資訊，請先將它們複製並保存在您的電腦筆記本中：Gemini API Key:前往 Google AI Studio。點擊「Get API key」建立並複製一組新的 API Key。ImgBB API Key:前往 ImgBB API 頁面 並註冊/登入。點擊「Add API key」建立並複製您的 API Key。LINE Channel Access Token 與 LINE User ID:前往 LINE Developers Console，登入您的 LINE 帳號。建立一個新的 Provider，然後建立一個 Messaging API channel。在 Channel 設定的 Messaging API 標籤頁底部，點擊「Issue」產生並複製 Channel access token (long-lived)。在 Channel 設定的 Basic settings 標籤頁底部，找到並複製您的 Your user ID（一串以 U 開頭的英數字）。重要：請拿出手機，掃描 Messaging API 標籤頁中的 QR Code，將您剛剛建立的官方帳號加入好友。步驟二：建立 GitHub 儲存庫 (Repository)登入您的 GitHub 帳號，點擊右上角 + -> New repository。命名為 daily-morning-bot（或您喜歡的名稱），設為 Private（私有），點擊 Create repository。步驟三：設定 GitHub Secrets (環境變數)為了安全地存放您的 API 密鑰，我們將把它們設定在 GitHub 儲存庫的安全設定中：進入您剛建立的 GitHub 儲存庫頁面。點擊上方的 Settings -> 左側邊欄展開 Secrets and variables -> 點擊 Actions。點擊綠色按鈕 New repository secret，依序新增以下四個 Secret（名稱必須完全一致）：Name: GEMINI_API_KEY, Value: (貼上您的 Gemini API Key)Name: IMGBB_API_KEY, Value: (貼上您的 ImgBB API Key)Name: LINE_CHANNEL_ACCESS_TOKEN, Value: (貼上您的 LINE Token)Name: LINE_USER_ID, Value: (貼上您的 LINE User ID)步驟四：上傳程式碼請將本回覆中提供的三個程式碼檔案，建立並上傳至您的 GitHub 儲存庫。結構必須如下：main.py (放在根目錄)requirements.txt (放在根目錄).github/workflows/daily_morning.yml (請務必建立 .github 資料夾，裡面建立 workflows 資料夾，再把 daily_morning.yml 放進去)步驟五：測試與自動排程手動測試：在 GitHub 儲存庫頁面，點擊上方的 Actions 標籤。點選左側的 Daily Morning Bot 工作流程。點擊右側的 Run workflow 綠色按鈕，執行手動測試。等待約 1~2 分鐘，檢查您的 LINE 是否收到早安圖！自動排程：目前的設定檔 daily_morning.yml 已設定為每天早上台灣時間 7:00 執行。只要 GitHub 保持運作，您每天都會準時收到滿滿正能量的早安圖。
